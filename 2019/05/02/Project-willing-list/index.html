这学期想要完成的项目和练习

发表于 2019-05-02 | 更新于 2018-03-18 | 阅读次数：

此文中列举了2017-2018学年第二学期，也就是2018年上半年我想要完成的任务，截止至8月底开学前。

现在未完成的任务

大创

这个真的是让人头大，做起来很繁琐，自己当初挖了大坑，主题是基于蓝牙的智能跟随机器人，目前正在申请专利中，实物模型还在制作，队员跟我感觉差不多，非常想要赶快结束这个任务。
大创大概4月底到5月中旬可以结束，4月底交材料五月中旬答辩。答辩结束后会把一部分底盘的控制代码分享。

蓝桥杯（4月1日初赛）

emmmm作为一个未来要跨考计算机专业的小白，见识一下比赛比较好，也督促一下自己学习学习数据结构和算法，为后面参加PAT认证考试打基础。
拿不拿奖倒是无所谓，但是要是没有好好准备，那是真的心疼交的300软妹币。

英语竞赛（4月20日左右）

又是上学期挖的坑，跪着也得完成。

现在将要进行的任务（规划）

电子设计竞赛（市级）

今年是偶数年，要准备一波模电和430单片机，虽然这学期刚开始学模电。

电赛国赛练习

明年还会有国赛，现在有些经验，可以做一些准备，打算选择控制类的题目，
所以要完成：

  • 平衡小车
  • 倒立摆或风力摆
  • 使用仿真或者其他计算方法快速调节PID参数

数学建模校赛

下学期会有数学建模国赛，数学建模可以学习很多东西，是一个我觉得性价比很高的比赛，也是一种磨练，这学期可能还是用深圳杯作为校赛。

日常练习

C/C++练习

用来积累代码经验的发育项目，跟以前一样，从基础开始快的话大概两周一练，考试周和考试前一周除外（考试要紧），所以这学期大概会有8-10个项目练习（加上电赛什么的项目）。
目前的项目：

  • 使用MFC的简易计算器
  • 上面那个计算器的升级版，表达式计算器（目测使用优先级列表对运算符使用栈操作）
  • 从某软件上下载电视节目（应该是涉及网络编程）
  • 自动删除微博的脚本
  • 坦克大战和海战棋游戏

打算学习的工具

  1. Qt
  2. Cocos
  3. Multisim
  4. IDEA、Pycharm
  5. 微信小程序开发
  6. Wireshark
  7. 上面的能学完就不错了= =

打算学习的编程语言？

  1. JAVA
  2. Python

  4. Javascript
  5. C# ?(以后Unity可能会用到)
    但是这学期能学完1-2个就不错了

为跨考做的准备

专业课

这学期要完成第一遍408专业课的学习，目前是第2周周末，已经完成第一遍计算机网络的初步学习，下一学习周期预计在7月底。
下一门要进行的课程是数据结构与算法，也是为了蓝桥杯和下学期的PAT做准备。预计第一遍完成时间：3月31日。第二遍应该是在暑假进行了。
4月蓝桥杯结束后进行计算机组成和操作系统的同步学习，并加入汇编语言的学习。

基础课

考研除了408以外，数学是最重要的，高数线代概率论都不能忘记，这学期电气的专业课对高数线代的要求也不小，甚至还有图论内容，正好可以复习复习，也把吉米多维奇的题和高数书多看几遍，这是最重要的，英语什么的我还算是凑合，要靠平时的单词积累。
